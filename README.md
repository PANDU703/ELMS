# ELMS - Employee Leave Management System

ELMS Pro is a full-stack Employee Leave Management System developed 
using Spring Boot, Spring Security, MySQL, Bootstrap, and JavaScript. 
It helps employees apply for leave and allows managers to review, approve, or reject requests through a secure and user-friendly interface. 
The system also includes email notifications, role-based authentication, dashboard analytics, and profile management to provide a smooth leave management experience. 
This project was developed as a capstone project to strengthen my skills in full-stack Java development and real-world application design.
